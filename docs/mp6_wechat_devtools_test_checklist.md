# MP6 微信开发者工具和真机测试清单

1. 首页可打开，文案说明清楚。
2. 学校库可打开。
3. 搜索学校名称和简称。
4. 区域筛选。
5. 类型筛选。
6. 住宿筛选。
7. 分数线状态筛选。
8. 多条件筛选同时生效。
9. 学校详情可打开。
10. 历史分数线正常展示或显示空状态。
11. 收藏学校。
12. 取消收藏。
13. 收藏页可打开。
14. 收藏页可清理无效收藏。
15. 目标记录保存。
16. 目标记录删除。
17. 清空目标记录。
18. 我的页清空本地数据。
19. 数据说明页可打开。
20. 隐私说明页可打开。
21. 我的页显示版本、学校数量、分数线数量。
22. 复制学校名称。
23. 复制地图搜索词。
24. 复制学校来源链接。
25. 复制分数线来源链接。
26. 检查没有登录入口。
27. 检查没有定位入口。
28. 检查没有支付入口。
29. 检查没有广告入口。
30. 检查没有录取预测入口。
31. 检查没有志愿推荐入口。
32. 检查控制台无红色报错。
