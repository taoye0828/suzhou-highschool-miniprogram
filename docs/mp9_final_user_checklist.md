# MP9 最终人工清单

提交审核前逐项确认：

1. AppID 已换成真实 AppID。
2. 编译通过。
3. Console 无红色报错。
4. 手机预览通过。
5. 学校库正常。
6. 历史分数线正常。
7. 收藏正常。
8. 目标记录正常。
9. 数据说明正常。
10. 隐私说明正常。
11. 我的页正常。
12. 无登录入口。
13. 无定位入口。
14. 无支付入口。
15. 无广告入口。
16. 无录取预测入口。
17. 已上传开发版本。
18. 已设置体验版。
19. 体验成员已复测。
20. 已提交审核。

建议在第 20 项提交审核前，保留一份编译通过和手机预览通过的截图或记录。若出现 Console 红色报错，先复制完整报错全文再处理。
