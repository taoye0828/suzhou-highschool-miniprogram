# MP6 用户手动步骤

1. 登录微信公众平台。
2. 确认小程序账号可用。
3. 确认小程序已备案。
4. 确认服务类目。
5. 获取真实 AppID。
6. 在微信开发者工具导入时填写 AppID，或只在本机 project.config.json 中填写 AppID。
7. 不填写 AppSecret。
8. 打开微信开发者工具。
9. 导入项目目录 `/Users/tom/Dev/suzhou_highschool_miniprogram`。
10. 选择不使用云服务。
11. 点击编译。
12. 真机预览。
13. 按 `docs/mp6_wechat_devtools_test_checklist.md` 检查页面。
14. 上传开发版本。
15. 设置体验版。
16. 添加体验成员。
17. 真机复测。
18. 使用 `docs/mp6_review_submission_notes.md` 准备审核备注。
19. 提交审核。
20. 审核通过后发布线上版本。

当前仓库不包含真实 AppID、AppSecret、token、密码或服务端密钥。
