# MP7 微信上传快速指南

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录填写：`/Users/tom/Dev/suzhou_highschool_miniprogram`
4. AppID 填微信公众平台里的真实 AppID。
5. 不使用云服务。
6. 点击“编译”。
7. 点击“预览”，用手机扫码测试。
8. 检查首页、学校库、学校详情、收藏、学习目标记录、数据说明、隐私说明、我的页。
9. 确认没有登录、定位、支付、广告、录取预测或志愿推荐入口。
10. 点击“上传”。
11. 版本号填：`1.4.0`
12. 版本描述建议：

```text
苏州高中目标查询助手 1.4.0：展示苏州高中阶段学校基础信息和官方来源可核验的 2025 年历史录取分数线；支持本地收藏和学习目标记录；不登录、不定位、不上传用户数据；不做录取预测，不提供志愿填报结论。
```

13. 打开微信公众平台。
14. 进入版本管理。
15. 将开发版本设置为体验版。
16. 添加体验成员。
17. 真机复测。
18. 使用 `docs/mp6_review_submission_notes.md` 或 `docs/mp7_final_handoff.md` 中的审核备注提交审核。
19. 审核通过后再发布线上版本。
