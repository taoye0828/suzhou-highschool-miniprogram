# MP6 审核备注建议

本小程序用于展示苏州高中公开基础信息，支持本地收藏和学习目标记录。当前不提供录取预测，不提供志愿填报结论，不根据用户分数推荐学校，不涉及登录、支付、广告、定位、用户上传或云端同步。收藏和学习目标记录仅保存在本机本地缓存。历史录取分数线如有展示，仅来自官方来源可核验数据，仅供了解，不代表未来录取结果。

当前版本展示 2025 年官方历史录取分数线。分数线数据来自苏州教育考试院官方公开页面，页面中已展示年份、招生区域、批次、招生类型、分数口径和来源说明。当前不展示 2026 年学校级分数线。
