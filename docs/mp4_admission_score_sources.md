# MP4 历史录取分数线来源说明

## 当前状态

- 历史录取分数线数量：0 条
- `data/admission-scores.js`：存在，`admissionScores` 为空数组
- 页面结构：已支持分数线展示、空状态和安全说明

## 未录入原因

本轮未录入具体分数线，是因为可直接读取的官方文本页面中未同时满足学校、年份、招生区域、批次、招生类型和分数口径的结构化要求。部分官方页面以图片展示分数线，当前未做 OCR 录入，避免把口径不清的数据写入正式页面。

## 后续录入条件

后续只有在教育局、招考部门、政府公开页面或学校官方招生公告中明确提供以下字段时才可录入：

- 年份
- 招生区域
- 批次
- 招生类型
- 分数口径
- 最低分或对应分数
- 同分规则（如有）
- 来源标题、链接和核对日期
